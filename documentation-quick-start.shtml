Quick Start Guide

Welcome to the Algobots Competition Platform! This guide will take you on the express lane to trading with your algobot live in 60 seconds. Once you are up and running, use the menu on the left to access the rest of the documentation that will help you make the most out of your algobot and the platform.

Requirements

  • A verified account with Poloniex. If you don't have one, get it here. Make sure you go through the verification process so that you can trade freely.

  • An API Key to access your account programmatically. If you don't have an API key, follow the instructions to get one.

  • A modern Internet browser like Chrome, Edge, Firefox or Safari.

Get Your Token to Access the Platform

You will need to come up with a name for your Team (other people may want to join your team later on, so pick a cool name), a name for your Algobot and your own UserName. Send those to us and you will get a URL to access the platform in return.

Access the Platform

Access the platform by navigating to the URL we sent you. Once the platform finishes loading, you will see something like this:

Notice the elements marked in red numbers on the top bar:

  1. Your user name.

  2. Your team's name.

  3. Your algobot's name.

  4. Your algobot's main process name (algobots may have more than one process).

  5. The running mode: Backtest / Live / Competition (click to cycle through all options).

  6. The Play button.

Access Your Algobot's Code

Set the running mode to Backtest and click the Play button on the top right corner of the screen. Then hit F12 in your keyboard to open your browser's Developer Tools and select the Source tab.

You will find the source code of your algobot in the User.Bot.js file under the Bots / AAYourTeam / members / YourUserName / AAYourAlgobot-Trading-Process > Trading-Process folder.

Edit the Source Code

Try editing the source code by including a simple comment line and hit Ctrl+S / Cmd+S to save the changes.

Trade Live

In order to trade live you will need to send us your API Key for us to configure the access to the exchange from the Cloud. Once the configuration is done, cycle to Live mode and click Play.

You may login to the exchange and check the My Open Orders and My Trading History & Analysis pages to find the orders placed by your algobot.

That's it!

Now that you are up and running, keep exploring this documentation to learn more about the platform and your algobot so that you can get the most out of them. We recommend starting with the Intro page!
