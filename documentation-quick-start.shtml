Quick Start Guide

Welcome to the Algobots Competition Platform! This guide will take you on the express lane to getting your algobot. Once you are up and running, use the menu on the left to access the rest of the documentation which should help you make the most out of your algobot and the platform.

Registration

  1. Make sure you are using Chrome (Version 67.0.3396.87 or later); it is the only tested browser at this point in time.

  2. Register in the Users' Portal. Feel free to complete your basic profile and upload an avatar picture.

  3. Now go to the Teams' Portal and create you own team. Later on, other people might want to join you in your efforts and you will be able to choose who to let in your team. Feel free to complete the team's basic profile and upload an avatar picture. Make sure you pick a name for your bot and upload an avatar picture for it as well.

Access the Platform

You are ready to go! Now go to the Development IDE and start playing around with your bot. Please be patient... it may take up to two minutes for the platform to load completely (you should see the progress on-screen). Once the platform finishes loading, you will see something like this:

Notice the elements marked in red numbers on the top bar:

  1. Your user name.

  2. Your team's name.

  3. Your algobot's name.

  4. Your algobot's main process name (algobots may have more than one process).

  5. The running mode: Backtest / Live / Competition (click to cycle through all options).

  6. The Play button.

Access Your Algobot's Code

Set the running mode to Backtest and click the Play button on the top right corner of the screen. Then hit F12 in your keyboard to open Chrome's Developer Tools and select the Source tab.

You will find the source code of your algobot in the User.Bot.js file under the Bots / AAYourTeam / members / YourUserName / AAYourAlgobot-Trading-Process > Trading-Process folder.

Edit the Source Code

Try editing the source code by including a simple comment line and hit Ctrl+S / Cmd+S to save the changes. Seconds after you save, CloudVM should log the following events:


CloudVM -> Bot code changes detected.
CloudVM -> Changes in code saved.

(find the logs at Chrome's Console)

That's it!

Before moving on, please refresh the page ( F5 ) so that the rest of the interface updates with your algobot's current configuration.

With this set up, you are ready to run you algobot in backtest mode. Trading live is currently only available in a competition setting. You should get a basic understanding of how your algobot and the platform work and join our Advanced Algos Telegram Group to learn more about competitions.

Now that you are up and running, keep exploring this documentation to learn more about the platform and your algobot so that you can get the most out of them. We recommend starting with the Interface page!

Competitions

When you are ready to compete in the ALGO Arena, you will need:

  • A verified account with Poloniex. If you don't have one, get it here. Make sure you go through the verification process so that you can trade freely. We recommend either creating a new account to use specifically with Advanced Algos, or at least make sure you have very little money in it—just enough to play around and participate in competitions (0.002 BTC is more than enough).

  • An API Key to access your account programmatically. If you don't have an API key, follow the instructions to get one.

  • Join our Advanced Algos Telegram Group to learn more about competitions.
